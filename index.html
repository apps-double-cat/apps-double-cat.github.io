Laravel shopify

Installation steps

Config shopify app
Config app url and white list url for your app
  • App url: https://{your-doamin}/shopify-install
  • Allowed redirection URL(s): https://{your-doamin}/authorize
Packages install
Install package to your laravel app
  • Add require package to composer.json file: appsdoublec/laravel-shopify: "package-version"
  • Add source package to repositories:
"repositories": [
    { 
        "type": "git",
        "url": "git@github.com:appsdoublec/laravel-shopify.git"
    }
]
  • Run composer install or composer update
